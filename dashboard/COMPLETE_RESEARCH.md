# 10. ROI CALCULATOR DATA

## Calculation Formula

```javascript
export function calculateROI(inputs) {
  const {
    monthlyCallVolume,
    currentAnswerRate,
    currentConversionRate,
    avgAppointmentValue,
    region,
    scenario = 'base',
  } = inputs
  
  // Current State
  const answerRate = currentAnswerRate / 100
  const conversionRate = currentConversionRate / 100
  const currentAppointments = monthlyCallVolume * answerRate * conversionRate
  
  // Missed calls (opportunity)
  const missedCalls = monthlyCallVolume * (1 - answerRate)
  
  // With AI
  const aiAnswerRate = 0.99
  const aiConversionRates = { conservative: 0.40, base: 0.50, optimistic: 0.60 }
  const aiConversion = aiConversionRates[scenario]
  
  // AI appointments
  const aiAppointments = monthlyCallVolume * aiAnswerRate * aiConversion
  const additionalAppointments = aiAppointments - currentAppointments
  const additionalRevenue = additionalAppointments * avgAppointmentValue
  
  // After-hours capture
  const afterHoursCalls = monthlyCallVolume * 0.25
  const afterHoursAppointments = afterHoursCalls * 0.55
  const afterHoursRevenue = afterHoursAppointments * avgAppointmentValue
  
  // AI Costs
  const aiTotalCost = (monthlyCallVolume * 0.0303) + 10
  
  // Net Savings
  const monthlyRevenue = additionalRevenue + afterHoursRevenue
  const monthlySavings = monthlyRevenue - aiTotalCost
  const annualSavings = monthlySavings * 12
  
  // ROI
  const roiPercent = (monthlySavings / aiTotalCost) * 100
  const paybackDays = (aiTotalCost / (monthlySavings / 30))
  
  return {
    monthlySavings,
    annualSavings,
    roiPercent,
    paybackDays,
    additionalAppointments,
    afterHoursAppointments,
  }
}
```

## Booking Windows
- **Emergency:** Same-day or next-day only
- **Regular:** 2-4 weeks out
- **New patient:** 1-2 weeks out (high cancel risk)

## Emergency Triage Questions (In Order)

```
1. "Where is the pain?" (tooth location)
2. "On a scale 1-10, how bad?" (8+ = urgent)
3. "How long has it been hurting?" (hours vs days)
4. "Can you bite down?" (mobility = urgent)
5. "Any swelling in face/jaw?" (infection = urgent)
```

### Emergency Handling Rules
| Condition | Action |
|-----------|--------|
| Pain 8-10 | Same-day (drop everything) |
| Swelling/fever | Same-day or ER |
| Trauma (broken tooth) | Same-day priority |
| Post-op complications | Contact dentist directly |

## New Patient vs Existing Patient

### New Patient Workflow
1. Verify insurance (if asked)
2. Collect: name, phone, email
3. Ask: "Do you have dental anxiety?"
4. Ask: "What brings you in today?"
5. Book **60-minute** appointment
6. Send consent forms via email BEFORE visit

### Existing Patient Workflow
1. Name/DOB lookup
2. Ask: "What brings you in?"
3. Book based on appointment type
4. Skip paperwork

**Key for AI:** `IF new_patient THEN ask_more_questions`

## Top 20 Patient Questions

### Insurance & Cost (Most Common)

**Q1: "Do you accept my insurance?"** ‚Üê MOST COMMON
- Ask: "What's your insurance carrier?"
- Check system ‚Üí "Yes, we're in-network" or "We can file your claim"
- **AI needs:** Insurance lookup function

**Q2: "What's the cost of [procedure]?"**
- Answer: "Typically $[range]. Recommend a consult for exact pricing"
- Counter: "We offer payment plans" / "Insurance covers 50-100%"

**Q3: "Do you take Medicaid/Medicare?"**
- Varies by practice - store as practice profile

**Q4: "Will my insurance cover this?"**
- Answer: "Let me verify. What's your plan?"
- **Never guess** (legal liability)

**Q5: "What if I don't have insurance?"**
- Answer: "We offer cash discounts" or "Payment plans available"

### Appointment Logistics

**Q6: "How long does the appointment take?"**
- "30-45 min for cleaning, 60+ for fillings"

**Q7: "Can I bring my kids?"**
- Varies by practice - store as policy

**Q8: "What time slots do you have?"**
- **THIS IS YOUR AI's MAIN JOB**
- Offer 3-5 specific options with dates/times

**Q9: "Can I reschedule if running late?"**
- "Yes, call if >10 min late"

**Q10: "Do I need to fast?"**
- "No, unless having sedation"

### Procedure Questions

**Q11: "Does it hurt?"**
- "No, we use local anesthesia. Pressure but no pain"

**Q12: "How many visits will I need?"**
- "Crown = 2 visits, implant = 3-6 months total"

**Q13: "Can you whiten my teeth?"**
- Check if practice offers it

**Q14: "Is my cavity urgent?"**
- "If painless, 1-2 weeks OK. If sensitive, sooner"

**Q15: "Do you do root canals?"**
- "Simple ones yes. Complex ones we refer to specialists"
