## Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Navy Blue | `#002855` | Primary brand color, trust/authority |
| Teal | `#1F6B7` | Text, secondary elements |
| Green | `#7AB800` | Growth arrow, success indicators |
| White | `#FFFFFF` | Phone icon, backgrounds |

## File Naming Convention

```
callguard-primary.png      → Main logo (landing page, hero)
callguard-favicon.png      → Favicon/app icon (16x16 to 64x64)
callguard-horizontal.png   → Header/signature (wide format)
callguard-white.png        → White version for dark backgrounds
callguard-icon-only.png    → Shield only, no text
```
