# Clinic Playbooks

Practice-type-specific configuration guides for DentSignal setup.

## Overview

Each playbook covers:
- **Ideal configuration** - Settings that work best for this practice type
- **Common scenarios** - Typical calls and how the AI should handle them
- **Scripts to customize** - Language specific to this specialty
- **Integration notes** - Practice management software considerations

## Playbook Index

| Practice Type | File | Key Differentiator |
|--------------|------|-------------------|
| Solo Practitioner | `solo-practice.md` | Flexible hours, personal touch |
| General/Family Practice | `general-practice.md` | High volume, diverse services |
| Pediatric Dentistry | `pediatric-practice.md` | Parent communication, kid-friendly |
| Specialty Practice | `specialty-practice.md` | Referral-based, complex scheduling |

---

## Quick Configuration Checklist

Every practice needs these configured:

- [ ] Business name and location
- [ ] Operating hours (including lunch breaks)
- [ ] Services offered (for scheduling)
- [ ] Emergency handling rules
- [ ] Transfer preferences
- [ ] After-hours behavior
- [ ] SMS confirmation settings
- [ ] Voicemail fallback (if desired)
