# DentSignal Sales Templates

Ready-to-use templates for dental clinic outreach. Copy, personalize, and send.

## Quick Start

1. **LinkedIn** → Start with connection request, follow up 2-3 days later
2. **Email** → Send cold email, follow up in 3 days if no response  
3. **Demo** → Use 10-min script for qualified leads

## Template Files

| File | Purpose | Best For |
|------|---------|----------|
| `linkedin-messages.md` | Connection requests, follow-ups | Clinic owners on LinkedIn |
| `cold-emails.md` | Email sequences | Clinic email addresses |
| `demo-script.md` | Live demo walkthrough | Scheduled meetings |
| `objection-handling.md` | Common objections + responses | All conversations |

## Target Personas

### 1. Solo Practitioner (1-2 dentists)
- **Pain:** Answering phones during procedures
- **Hook:** "What if you never missed a call again?"
- **Price:** Starter ($149/mo)

### 2. Multi-Location (3+ offices)
- **Pain:** Inconsistent patient experience
- **Hook:** "Same great service at every location, 24/7"
- **Price:** Enterprise ($399+/mo)

### 3. Office Manager / Front Desk
- **Pain:** Juggling phones + patients + scheduling
- **Hook:** "Focus on in-person patients, AI handles calls"
- **Price:** Growth ($199/mo)
